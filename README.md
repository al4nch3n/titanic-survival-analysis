# Titanic Survival Rate Analysis
This project uses pandas to analyze the Titanic database to explore whether certain factors influenced survival rate of passengers.
## Tools:
- Python
- Pandas
- Numpy
- Matplotlib
- Jupyter Notebook
- Anaconda Navigator
## What I Did:
- Checked dataset for missing values
- Calculated survival rates by gender and passenger class
- Calculated survival rates across gender and class together
- Created a family size feature using relatives recorded aboard
- Grouped passengers into family sizes and calculated survival rates
- Displayed findings using a heat graph on matplotlib
## Key Findings:
- Female passengers had a much higher survival rate than male passengers.
- First class passengers had a higher survival rate than second and third class passengers.
- Passengers travelling in small families (2-4 members) had the highest survival rates.
- Passengers travelling solo or in large families had lower survival rates.
